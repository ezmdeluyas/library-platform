# Library Platform

Library Platform is a production-style microservices-based library lending system designed to demonstrate real-world backend architecture using Spring Boot, JWT authentication, PostgreSQL, and Docker.

The system consists of:

- Auth Service (Authentication & Token Management)
- Library Service (Catalog & Lending Domain)
- Next.js Frontend (User Interface)
- PostgreSQL (Data Persistence)

This project showcases:

- Access & Refresh Token flow
- Role-based Authorization
- Clean service separation
- RESTful API design
- Dockerized multi-service architecture
- Database migrations with Flyway
- Production-ready configuration practices

This repository is structured as a monorepo containing all services and infrastructure configuration.
